# 厚労省発表と自治体発表の死亡者数の比較
新型コロナウイルス(COVID-19)感染者数の統計について、厚生労働省は突合作業が完了した死亡者数のみを発表しているため、自治体発表の死亡者数とずれが生じています。このチャートでは、それぞれの発表の死亡者数を比較します。
